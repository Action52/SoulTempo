Nosotros

Soul Tempo es un software libre basado en la idea de que el arte crea arte.
Este software se dedica a procesar imágenes a través de un algoritmo que le permite generar un sonido apartir de las mismas

¿Cómo Funciona?

El algoritmo de Soul Tempo basado en Match Learning segmenta la imagen a procesar y calacula la concentración, intensidad y el color para poder generar una nota cuya duración y volumen dependen del color evaluado.
Soul Tempo tiene la hablidad de generar notas las cuales pueden llegar a crear una melodía con cierto grado de complejidad.
